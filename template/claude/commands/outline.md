# Outline Section

Generate a structured outline for a paper section before writing full prose.

## Instructions

This is **Stage 1** of the two-stage writing process. The user specifies which section to outline.

1. **Read `main.tex`** to understand the paper's current state, topic, and existing sections
2. **Research** the topic area using `perplexity::search` or `research-lookup` to gather:
   - Key concepts and terminology
   - Important references to cite
   - Standard structure for this type of section
3. **Generate an outline** with:
   - Subsection headings (if appropriate)
   - 3-5 key points per subsection, each with:
     - The core claim or finding
     - Supporting evidence or citation keys
     - Connection to the paper's thesis
   - Suggested figures or tables
   - Transition notes between subsections
4. **Present the outline** for user approval before converting to prose
5. After approval, convert to full paragraphs and edit into `main.tex`
